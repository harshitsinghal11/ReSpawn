# Contributing to ReSpawn

Thank you for your interest in contributing to ReSpawn.

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test your changes
5. Submit a Pull Request

## Development Requirements

* Windows 10/11
* .NET 8 SDK
* Visual Studio 2022 or JetBrains Rider

## Coding Guidelines

* Follow existing project structure and naming conventions
* Keep code clean and maintainable
* Write descriptive commit messages
* Avoid unnecessary dependencies

## Reporting Bugs

When reporting a bug, please include:

* ReSpawn version
* Windows version
* Steps to reproduce
* Expected behavior
* Actual behavior

## Feature Requests

Feature requests are welcome. Please open an issue and clearly explain the proposed improvement and its use case.
